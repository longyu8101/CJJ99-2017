 <ol>
 <li>桥上应设置交通信息显示屏。</li>
 <li>雾天桥上行车时速宜符合<a href="#B3.0.18.1">表3.0.18-1</a>的规定。</li>
 </ol>


 <ol strat="3">
 <li>当风速大于19 m/s时，桥上行车时速宜符合<a href="#B3.0.18.2">表3.0.18-2</a>的规定。当风速大于21 m/s时，严禁货运车上桥行驶。当悬索有明显振颤时，应在现场进行监视或录像，并应进行记录。</li>
 </ol>
